# Web Dev 1

## Overview
A simple web development starter project using Node.js and Express.

## Project Structure
- `server.js` - Express server running on port 5000
- `public/` - Static files directory
  - `index.html` - Main HTML page
  - `styles.css` - Stylesheet
  - `script.js` - Client-side JavaScript

## Running the Project
```bash
npm start
```

## Technology Stack
- Node.js 20
- Express.js 4.x
